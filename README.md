# Homework 3: JIT Compilation and Tracing for ML on GPUs

This repository contains my submission for Homework 3 in DSP for AI.

## Files
- `jax_jit_analysis.ipynb` — JAX JIT analysis
- `torch_compile_analysis.ipynb` — PyTorch `torch.compile` analysis
- `Tianyang.pdf` — final report

## Requirements
Recommended environment:
- Python 3.10+
- JAX / jaxlib
- PyTorch
- NumPy
- Matplotlib

## How to run
1. Open `jax_jit_analysis.ipynb` and run all cells in order.
2. Open `torch_compile_analysis.ipynb` and run all cells in order.
3. GPU is recommended for profiling sections.

## Notes
- Some profiling outputs depend on CUDA availability.
- The PDF report summarizes the main findings and results.
